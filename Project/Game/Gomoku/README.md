# 五子棋 Gomoku 🎲

一款开箱即用的五子棋双人/人机对战网页，采用纯前端技术（HTML + CSS + JavaScript）实现。棋盘为15×15标准规格，界面精美，支持多主题、多语言和AI对战。

🔗 **在线试玩**：https://yang-2026-user.github.io/My-Repository/Project/Game/Gomoku/

📦 **仓库地址**：https://github.com/Yang-2026-user/My-Repository

---

## ✨ 功能特点

- **双人对战模式**：两位玩家轮流落子，适合朋友面对面切磋。
- **人机对战模式**：内置AI引擎，支持**简单 / 普通 / 困难**三档难度，让你随时挑战电脑。
- **智能悔棋系统**：
  - 双人模式：可向对方请求悔棋（需确认）。
  - 人机模式：玩家可主动悔棋，AI会根据当前局势和难度判断是否同意（困难模式下AI会“评估”局势后再决定）。
- **多主题切换**：内置 **榧木 / 深蓝 / 翠绿 / 石墨** 四种棋盘主题，视觉风格随心换。
- **多语言支持**：界面文字可自由切换 **中文 / English**，满足不同用户需求。
- **音效反馈**：落子、获胜、失败均有提示音，可一键开关。
- **规则完整**：严格遵循五子棋标准规则（先连成五子者获胜，棋盘下满判和）。
- **响应式设计**：完美适配手机、平板和桌面，触屏操作流畅。

## 🧠 AI 难度说明

| 难度   | 特点                                                         |
|--------|--------------------------------------------------------------|
| 简单   | 基于启发式评分，走子较快，适合新手入门                       |
| 普通   | 在简单基础上增加简单搜索（Minimax），攻守更均衡              |
| 困难   | 带4层搜索的Minimax + Alpha-Beta剪枝，具备较强棋力，适合进阶玩家 |

> 💡 AI在困难模式下会优先堵截玩家的“活四”和“双三”，并尝试构建自己的必胜局面。

## 🎮 操作方式

- **点击落子**：点击或触摸棋盘交叉点即可落子（当前回合有效）。
- **悔棋**：点击“悔棋”按钮，双人模式下会弹出确认框；人机模式下AI会根据局势决定是否同意。
- **重新开始**：点击“重新开始”按钮重置对局。
- **切换模式/执子/难度**：在顶部控制栏自由调整，切换时会自动重置游戏。
- **切换主题**：在“主题”栏选择你喜欢的棋盘风格。
- **音效开关**：点击 🔊 / 🔇 按钮切换音效。
- **中英切换**：点击“中文 / English”按钮一键切换界面语言。

🛠️ 技术实现要点

· 棋盘渲染：使用 Canvas 绘制，自适应屏幕大小，支持高清屏（DPR）。
· AI 引擎：基于 启发式评分 + Minimax 搜索（困难模式下带 Alpha-Beta 剪枝），通过评估函数和走子候选生成来实现智能决策。
· 多语言系统：内置 I18N 对象，支持中英文界面动态切换，所有文字均通过 data-i18n 属性管理。
· 多主题系统：通过 CSS 自定义属性（--board-a、--board-b 等）实现主题切换，无需重绘逻辑。
· 音效系统：利用 Web Audio API 生成简单的落子、胜负提示音，无需外部音频文件。
· 状态管理：清晰记录游戏状态、历史记录、当前玩家、胜负判定等，便于悔棋和重开。
