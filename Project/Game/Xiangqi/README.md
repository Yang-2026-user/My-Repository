# 中国象棋 · Chinese Chess 🏯

一个开箱即用的中国象棋双人/人机对战网页，采用纯前端技术（HTML + CSS + JavaScript）实现。你可以直接在浏览器中运行，无需任何后端依赖。

🔗 **在线试玩**：https://yang-2026-user.github.io/My-Repository/Project/Game/Xiangqi/ （若已开启 Pages）

📦 **仓库地址**：https://github.com/Yang-2026-user/My-Repository

## ✨ 功能特点

- **双人模式**：两位玩家轮流落子，适合朋友面对面切磋。
- **人机模式**：内置 AI 引擎（支持 4 级难度：简单 / 普通 / 困难 / 大师），让你随时挑战电脑。
- **悔棋系统**：
  - 双人模式：可向对方请求悔棋（需对方确认）。
  - 人机模式：玩家可主动悔棋，AI 会根据难度决定是否同意（困难/大师级 AI 会“评估”局势）。
- **音效反馈**：走子、吃子、将军、胜负均有提示音，可一键开关。
- **中英文切换**：界面文字和棋子名称可自由切换（中文/英文）。
- **规则完整**：包含将军、应将、将帅对面、困毙、长将判和、50回合自然限着等规则。
- **响应式布局**：适配手机、平板和桌面，触屏操作友好。

## 🧠 AI 难度说明

| 难度   | 搜索深度 | 思考时间上限 | 特点                         |
|--------|----------|--------------|------------------------------|
| 简单   | 1 层     | 300ms        | 极快，适合新手               |
| 普通   | 3 层     | 1000ms       | 平衡，适合休闲玩家           |
| 困难   | 5 层     | 4000ms       | 较强，会使用开局库和杀手启发 |
| 大师   | 8 层     | 9000ms       | 最强，深度迭代加深           |

> 💡 AI 在困难/大师模式下还会参考**开局库**，并具备**历史启发**与**置换表**（Transposition Table）加速搜索。

## 🎮 操作方式

- **点击棋子**：选中己方棋子（高亮显示）。
- **再次点击**：若点击的是合法目标（红/绿点标记），则执行走子；若点击的是己方另一棋子，则切换选中。
- **悔棋**：点击“悔棋”按钮，双人模式下会弹出确认框；人机模式下直接回退一步（AI 可能拒绝）。
- **重新开始**：点击“重新开始”按钮重置对局。
- **音效开关**：点击 🔊 / 🔇 按钮切换音效。

## 🚀 本地运行

1. 克隆仓库：

   git clone https://github.com/Yang-2026-user/My-Repository.git

3. 进入目录，直接用浏览器打开 index.html 即可：

   cd My-Repository
   open index.html   # 或双击文件

无需安装任何工具，无需启动服务器。

🛠️ 技术实现要点

· 棋盘渲染：使用 Canvas 绘制，自适应屏幕。
· AI 引擎：基于 Negamax 搜索 + Alpha-Beta 剪枝，配合杀手启发、历史启发、置换表、开局库。
· 局面去重：采用 Zobrist 哈希检测重复局面（长将判和）。
· 语言包：内置 I18N 对象，支持中英文界面及棋子名称切换。
