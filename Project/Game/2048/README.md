## ✨ 核心特性 · Core Features

- 🎯 **经典玩法** · Classic gameplay —— 方向键 / 滑动手势移动方块，相同数字合并  
  *Use arrow keys or swipe to move tiles, merge identical numbers.*
- 🏆 **本地最高分** · Local Best Score —— 自动保存最高分到浏览器 `localStorage`  
  *Automatically saves your best score to browser localStorage.*
- 🔊 **音效反馈** · Sound Feedback —— 移动、合并、胜利/结束均有音效提示  
  *Sound effects for moves, merges, win and game-over events.*
- 📱 **移动端适配** · Mobile Optimized —— 完美支持触摸滑动，响应式布局  
  *Fully supports touch gestures with responsive design.*
- 🎨 **流畅动画** · Smooth Animations —— 方块弹出、合并缩放动画，体验丝滑  
  *Tile pop and merge animations for a polished experience.*
- 🧩 **纯前端** · Pure Frontend —— 单 HTML 文件，无需安装，开箱即玩  
  *Single HTML file, no installation needed — play right away.*

## 🎮 游戏指南 · Game Guide

| 操作 · Action | 说明 · Description |
|---|---|
| ⬆ ⬇ ⬅ ➡ | 键盘方向键移动 · Arrow keys to move |
| 👆 滑动 · Swipe | 在游戏区域滑动屏幕 · Swipe on the game board |
| 🔄 合并 · Merge | 两个相同数字相遇时合并为它们的和 · Two identical tiles merge into their sum |
| 🎯 目标 · Goal | 拼出 **2048** 方块！ · Reach the **2048** tile! |
| 🏁 结束 · Game Over | 无可合并方块时结束 · Game ends when no moves remain |

> 💡 **小贴士** · Tip：尽量把大数字固定在角落，采用"蛇形"策略更高效！  
> *Keep your largest tile in a corner and use a "snake" strategy for better results!*
